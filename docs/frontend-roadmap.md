# Frontend Roadmap

## Phase 1

- Open the Unity project in `mobile/`
- Generate local Unity metadata and project state
- Attach bootstrap scripts to an initial scene
- Validate environment loading from `StreamingAssets/config/dev-api.json`
- Prefer `10.211.55.22` as the dev backend and keep localhost as fallback

## Phase 2

- Build a home screen with:
  - mascot area
  - child overview
  - activities list
  - rewards summary
- Perform `dev-login` automatically in development mode
- Show progress summary and assets catalog

## Phase 3

- Add activity detail flow
- Add progress check-in action
- Refresh rewards and progress after check-in
- Import Gau `.fbx` and place the mascot in the bootstrap scene

## Phase 4

- Replace text-only presenters with real Unity UI
- Prepare transition from dashboard to 3D activity scenes
